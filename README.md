Code and material for FAIR Enough Hands-on Workshop at SFSCON2025

Usage:

1. run create_structure.sh <nparticipants>
2. run create_subj_table.sh <nparticipants> > <target_path>/participants.tsv
3. run create_json_sidecars.sh <target_path>/dataset_description.json <target_path>/task-FAIRevents.json
